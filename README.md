2.  基于Html5规范，其中包括：

    a.  引用html5标准的header、footer等语义化标签。

    b.  引用html5标准del等设计标签。

    c.  将原不规范的html，重设为规范化，旧头部为：

> \<html lang=\"zh-CN\"\>
>
> \<head\>
>
> 格式化后的标准：
>
> \<!doctype html\>
>
> \<html lang=\"zh-CN\"\>
>
> \<head\>

> \<meta charset=\"utf-8\"\>
